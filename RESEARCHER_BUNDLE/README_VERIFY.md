# Assembly Theory Researcher Bundle

This is a self-contained Lean 4 package for verifying the Assembly Theory formalization.

## Quick Start

```bash
# Verify everything in one command
./scripts/verify_assembly.sh
```

## Manual Verification

### 1. Build the Project

```bash
lake update
lake build
```

### 2. Check for Proof Gaps

```bash
grep -rn "sorry\|admit" HeytingLean/
# Expected: no output
```

### 3. Strict Build

```bash
lake build -- -DwarningAsError=true
```

## What's Verified

| Theorem | File | Description |
|---------|------|-------------|
| `assemblyIndex_eq_dagJoinCount` | AssemblyIndex.lean | AI = reuse-aware join count |
| `assemblyIndex_ge_log2` | AssemblyBounds.lean | Lower bound: log2(size) |
| `assemblyIndex_le_size_sub_one` | AssemblyBounds.lean | Upper bound: size - 1 |
| `assemblyIndex_quotient_le` | AssemblyQuotient.lean | Quotient monotonicity |
| `assemblyIndex_mol_le_dagJoinCount` | MolecularSpace.lean | Molecular assembly bound |

## Requirements

- Lean 4 (version in `lean-toolchain`)
- Network access for fetching mathlib

## Troubleshooting

### "package not found"

Run `lake update` first.

### Out of memory

Use `lake build -j1` for single-threaded build.

### Wrong Lean version

```bash
elan install $(cat lean-toolchain)
```
